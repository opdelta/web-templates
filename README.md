# INF3190_TP1

## Description
Site web conçu dans le cadre du cours INF3190 à la session d'automne 2020.

Wikiplane est un site web dont l'objectif est de répertorier le plus d'avions possible pour créer un "Wikipedia" d'avions.

## Auteur

Ziad Lteif
Code permanent: LTEZ18059802

## Installation

Afin de faire fonctionner le site web comme il faut, il faut avoir un serveur PHP (EasyPHP fonctionne bien).

Il suffit de démarrer le serveur PHP avec tous les fichiers présents dans le dépot. 

## Informations sur les fichiers

```css```: Le dossier ```css``` contient toutes les feuilles de styles utilisées par le site web.

```files```: Le dossier ```files``` contient le fichier plainte.txt (créé si absent) qui est utilisé pour la sauvegarde des commentaires dans le formulaire.

```images```: Le dossier ```images``` contient toutes les images utilisées dans le site web. 

```php```: Le dossier ```php``` contient tous les fichiers ```.php``` servant au back-end du site web.

```README.md```: Ce document.

```about.html```: Page web qui explique Wikiplane.

```contact.html```: Page contenant le formulaire pour envoyer des commentaires (ou des plaintes!).

```index.php```: Page d'accueil contenant les articles de quelques avions populaire. 

```sources.html```: Vu que la plupart des images et des informations présent dans le site web ne sont pas libres de droits d'auteurs, la page ```sources``` contient toutes les sources utilisées.

```success.html```: Page de succès lorsque les commentaires (ou les plaintes!!) sont envoyés.

```toplist.php```: Page contenant un top 10 des avions les plus produits encore en production à ce jour.
