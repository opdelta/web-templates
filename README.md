## Repository contents

- tp2 branch: Contains a mock website for a school catering service that lets "parents" order from a set menu. Technologies used in this sample: CSS, HTML, PHP. 

- TP3 / main branches: Contains a mock website for an animal adoption website featuring a really simple and raw search engine. Technologies used in this mock website: CSS, HTML, PHP, JS and Bootstrap.

## Installation

In order to get started and load up any of the samples, you will need a PHP server (https://www.easyphp.org/). EasyPHP is a great tool easy to use and install.

1. Launch your PHP server.

2. Maintain the current folder structure and add the files to your server's directory.

3. By default, you can access the webpage using localhost or any other IP address in your browser.
